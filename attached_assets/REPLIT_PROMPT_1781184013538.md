## PROJECT: CubieTV — Android TV Media Player (UI Scaffold Only)

Build a complete **Android TV / Fire TV app UI scaffold** in Kotlin using **Jetpack Compose for TV**. This is a Netflix-style media player that browses a home NAS (SMB/DLNA) and plays videos via VLC. For this task, build **only the UI shell** — screens, navigation, components, theme, placeholder data. No actual VLC/SMB/network logic needed.

## Key Constraints (Read Before Writing Any Code)

1. **Compose for TV ONLY.** Never use `Fragment`, `BrowseFragment`, `PresenterSelector`, or any Leanback XML. Every screen is a `@Composable` function.

2. **D-pad is the ONLY input.** Every element reachable via Up/Down/Left/Right/Select. No swipe, no long-press-only actions. Test by tabbing through every screen mentally.

3. **Focus must always exist.** Every screen sets initial focus explicitly. Use `FocusRequester` + `LaunchedEffect` to request focus on first composition.

4. **No actual networking.** This is UI-only. Use `FakeData.kt` for all media items. Network code (libVLC, jcifs-ng, jUPnP) will be added later.

5. **Animations must be smooth.** Use `animateFloatAsState` for scale, `AnimatedVisibility` for overlays, `crossfade` for backdrop transitions.

6. **No placeholder images from the internet.** Use solid-color `Box` composables with the item's color + title text centered. Coil is imported but not used for placeholder art.

7. **Back button works everywhere.** Handle `BackHandler` in screens that need custom back behavior (player, search keyboard).

8. **Keep files under 500 lines.** Split large screens into sub-composables.
